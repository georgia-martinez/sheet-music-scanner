# Sheet Music Scanner

### Setup


Create the environment

`python3 -m venv .venv`

Activate the environment

`source .venv/bin/activate`

Install requirements

Download node

https://nodejs.org/en

Download poppler 

`brew install poppler`
`
`pip3 install -r requirements.txt`

### Run
`python3 app.py`

`cd frontend`
`npm install`
`npm start`
